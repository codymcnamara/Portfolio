Recent Work

Grumpypants

A Tumblr clone exclusively for complaining

  • Uses custom SQL query and overwrites Backbone’s “parse”method to order blog posts by most recently created and allow page pagination
  • Stores blogs and posts in Backbone collections to avoid unnecessary database querying
  • Uses filepicker library to support user photo upload on Backbone views
  • Implements custom Backbone “CompositeView” class that recursively removes blog or post subviews

Chess

A Chess game built in ruby to played at the terminal

  • Custom recursive method “deep_dup” copies the game board to check if move is valid and/or puts other player in check
  • Chess pieces inherit from parent classes where piece movements are similar (ex: bishops and rooks both inerit from “Slidable” class)

Pong

The game of Pong, built for the browser

  • Creates a Pong namespace to organize application logic in different files and make program more object oriented
  • Sharpness of angle at which ball travels after hitting paddle determined by distance from center of paddle

SolarCub

An app to help homeowners make informed solar purchases

  • Uses googlemaps API to create an input box for users to easily select their city. This allows the application to accurately capture geographic coordinates for those cities
  • Sends form data in ajax query to PV Watts API in order to generate accurate estimate of annual kilowatt hour production
